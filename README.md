# Portfolio Website – Yash Verma

A responsive personal portfolio website built using **HTML** and **CSS**.  
This project is created as part of the Mini Project for the course **Computer Science Fundamentals & Career Pathways**.



## Features

- Clean and modern user interface  
- Responsive layout (works on mobile and desktop)  
- About section  
- Skills section  
- Projects section  
- Contact form (demo)  
- Well-organized folder structure  
- Easy navigation
